# OmegaTester

## Run from the sources.
Prerequisites:
JDK 8
Maven > 3.2.1
Your favorite IDE

```
cd OmegaTester
mvn clean install
mvn exec:java
```

To change port use the following commands.

```
cd OmegaTester
mvn exec:java -Dexec.args="8080"

```
